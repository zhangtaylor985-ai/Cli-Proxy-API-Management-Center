# Entrypoints

Start here when orienting in the repo:

- `src/main.tsx`
- `src/App.tsx`
- `src/router/MainRoutes.tsx`
- `src/pages/DashboardPage.tsx`
- `src/pages/LoginPage.tsx`
- `src/styles/variables.scss`
- `src/styles/layout.scss`
- `src/styles/themes.scss`

For page-level work:

- inspect the target page `.tsx`
- inspect the paired `.module.scss`
- inspect any related store or hook only if the page already depends on it
